Immediately Invoked Function Expression (IIFE) is a function that is
      defined and executed immediately after it is created.
    
      It is a common JavaScript pattern used to create a new scope and avoid
      polluting the global namespace.
      The syntax for an IIFE is as follows:
    
        Syntax:
        (function() {
        // code to be executed
        })();
    
        Example:
        (function() {
            console.log("Hello, I am an IIFE!");
        })();

        Output: Hello, I am an IIFE!
