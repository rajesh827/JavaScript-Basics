The event object in JavaScript is an object that is automatically passed to event handler functions when an event occurs. 
        It contains information about the event, such as the type of event, the target element, and any additional data related to the event.

        Common properties of the event object include:
        1. type: The type of event (e.g., "click", "keydown").
        2. target: The element that triggered the event.
        3. key: The key that was pressed (for keyboard events).
        4. clientX and clientY: The coordinates of the mouse pointer (for mouse events).
