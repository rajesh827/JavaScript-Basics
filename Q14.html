A dialog box in JavaScript is a pop-up window that allows you to interact with the user.
        It's commonly used for showing messages, asking for confirmation, and taking input.

        JavaScript provides three types of dialog boxes:
        1. alert(): Displays a message and an OK button.
        2. confirm(): Displays a message with OK and Cancel buttons, returning true or false.
        3. prompt(): Displays a message with an input field and OK/Cancel buttons, returning the input or null.
